# E-Commerce Sales & Customer Churn Analysis

## 📊 Project Overview
This project analyzes e-commerce sales data and customer churn behavior to identify key trends and improve customer retention strategies.

## 🔧 Tools Used
- SQL(Data Analysis)
- Power BI(Data Visualization)

## Dataset


## 📈 Key Metrics
- Total Revenue: 45M
- Total Orders: 16K
- Total Customers: 3K
- Churn Rate: 41%
- Revenue Lost due to Churn: 18M

## 💡Key Insights
- Senior customers contribute the highest revenue
- High churn observed in specific product categories
- Sales peak during certain months and decline afterward
- Significant revenue loss due to customer churn
